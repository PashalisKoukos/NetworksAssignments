# Network Infrastructure & Web Deployment (Cisco Packet Tracer)

A comprehensive networking project developed for the **Computer Networks Laboratory** at Harokopio University. This project demonstrates the design and implementation of a complex network topology, integrating dynamic routing and web services.

## 🌐 Project Overview
The goal was to create a functional multi-area network that simulates a real-world corporate environment (Headquarters and Branch offices) communicating over a routed infrastructure.

##  Key Features
- **OSPF Dynamic Routing:** Implementation of multi-area OSPF (Area 1 & Area 2) to ensure seamless communication between different geographical locations.
- **Network Topology:** Designed with Cisco 2911 routers, switches, and end-devices using Cisco Packet Tracer.
- **Web Service Integration:** Deployment of HTTP/HTML servers within the network, accessible across different OSPF areas.
- **Redundancy & Scalability:** Configured to handle potential link failures and support the addition of new network nodes.

##  Technical Specifications
- **Protocols:** OSPF (Open Shortest Path First), IPv4 Addressing, ICMP.
- **Services:** HTTP Server, DNS configuration.
- **Hardware Simulation:** Cisco ISR 2911 Routers, Generic Switches, PT Servers.
- **Web Stack:** HTML5/CSS3 for the internal web application.

##  Included Scenarios
1. **Static Routing:** Initial phase implementing manual route definitions.
2. **Dynamic Routing:** Migration to OSPF for automated route discovery and optimization.
3. **Web Application:** A functional multi-page HTML form and information site hosted on internal servers.
